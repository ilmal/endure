# Animal Icons Directory

This directory contains SVG icons from The Noun Project.

## Download Instructions

Visit The Noun Project and download SVG files for each animal. Save them with these exact filenames:

### Required Icons:

1. `bear.svg` - For: Björn
2. `wolf.svg` - For: Varg
3. `fox.svg` - For: Räv, Fjällräv
4. `moose.svg` - For: Älg
5. `deer.svg` - For: Rådjur, Ren
6. `boar.svg` - For: Vildsvin
7. `lynx.svg` - For: Lodjur
8. `badger.svg` - For: Grävling, Järv
9. `beaver.svg` - For: Bäver
10. `otter.svg` - For: Utter, Mink
11. `hare.svg` - For: Hare, Fälthare
12. `owl.svg` - For: Fjälluggla, Berguv
13. `eagle.svg` - For: Havsörn
14. `bird.svg` - For: Ripa, Tjäder, Orre, Trana
15. `duck.svg` - For: Grågås, Ejder
16. `paw.svg` - For: Default/Unknown
